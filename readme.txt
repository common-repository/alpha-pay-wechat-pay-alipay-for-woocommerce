== Installation ==

* In Wordpress -> Plugins menu, click “Add New”.  In the search field, search by “AlphaPay - WeChat Pay, Alipay & UnionPay for WooCommerce”.
* Install and activate the plugin.
* Go to AlphaPay' Website and [apply a merchant account](https://www.alphapay.com/contact-us/).
* After receive account information, you should enter Partner Code and Gateway Credential.
* Enable WeChat Pay, Alipay and UnionPay.

== Frequently Asked Questions ==

= The order showing on WooCommerce is not paid, but the customer has already paid with WeChat Pay, Alipay or UnionPay. Where should I check the order status? =
1. Login the [AlphaPay Dashboard](https://pay.alphapay.ca) with your Login ID and password.
2. In WooCommerce Order Details, find the alphapay_order_id and copy this number.
3. In AlphaPay BackOffice, search the order_id you found in WooCommerce Order Details and check the order status.
4. If Pay Success in AlphaPay BackOffice, change the order status in WooCommerce.

If you need more help, contact us 1-855-937-7888


= Does this require an SSL certificate? =

Yes! an SSL certificate must be installed on your site to use AlphaPay (WeChat Pay, Alipay and UnionPay) which make sure payment complete under secure circumstance.
